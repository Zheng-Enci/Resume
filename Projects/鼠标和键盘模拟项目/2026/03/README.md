# 鼠标和键盘模拟项目 - 2026年3月开发

## 开发概述
2026年3月对鼠标和键盘模拟项目进行了开发，主要实现了USB HID设备配置和鼠标控制API服务，支持RESTful API控制鼠标移动。

## 主要工作
- **USB HID设备配置**：编写完整的USB HID设备配置文档和脚本
- **鼠标控制API**：实现FastAPI接口，支持鼠标任意方向移动
- **代码重构优化**：重构代码结构，创建services文件夹，添加类型注释
- **依赖管理**：创建requirements.txt和install_requirements.py脚本

## 技术栈
FastAPI + Python + USB Gadget HID + Linux内核配置

## 开发时间
2026年3月

## 项目成果
- **USB HID配置文档**：编写完整的USB HID设备配置文档，涵盖内核检查、OTG模式切换、设备绑定等
- **鼠标API开发**：实现基于FastAPI的鼠标控制API，支持x、y坐标控制
- **代码质量优化**：重构代码结构，添加类型注释，创建统一响应类
- **开源贡献**：提供完整的配置文档和API服务
