# 鼠标和键盘模拟项目 - 2026年3月开发

## 开发概述
2026年3月对鼠标和键盘模拟项目进行了初始化和开发，主要实现了基于USB Gadget HID协议的鼠标控制API和键盘服务，支持RESTful API控制鼠标移动和键盘输入。

## 主要工作
- **项目初始化**：实现鼠标API服务、键盘服务、USB HID设备配置
- **鼠标控制API**：实现FastAPI接口，支持鼠标任意方向移动
- **键盘服务**：实现Flask微服务，支持短按/长按按键和字符串发送
- **USB HID配置**：编写完整的USB HID设备配置脚本和文档

## 技术栈
FastAPI + Flask + Python + USB Gadget HID + Linux内核配置

## 开发时间
2026年3月

## 项目成果
- **独立开发USB HID项目**：实现完整的鼠标和键盘模拟功能，支持RESTful API控制
- **开源贡献**：提供完整的配置脚本和文档，支持RK3588等Linux开发板
- **实用工具价值**：简化USB HID设备配置，提供统一的API接口，支持远程控制
