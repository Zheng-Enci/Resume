# AI坊学生管理系统 - 后端开发记录总览

## 项目概述
AI坊学生管理系统是厦门工学院人工智能创作坊的学生管理解决方案，后端部分采用Spring Boot技术栈开发，提供稳定高效的API服务和数据支撑。

## 技术架构

### 后端技术栈
- **框架**：Spring Boot 3.5.5（Java 24）
- **数据访问**：Spring Data JPA + Hibernate
- **安全认证**：Spring Security + JWT
- **数据存储**：MySQL 8.0 + Redis
- **缓存优化**：Redis缓存签到状态，响应时间优化至100ms内
- **文件存储**：MinIO（学生打卡照片存储）
- **接口文档**：Knife4j 4.6.0

### 核心功能模块
1. **学生信息管理模块**：学生注册、登录、信息CRUD操作
2. **分时段签到系统**：早/午/晚三个时段签到控制，防重复机制
3. **安全认证模块**：JWT Token管理，IP白名单限制
4. **数据统计模块**：多维度数据分析与统计查询
5. **积分管理模块**：积分记录创建、排名查询、改分记录
6. **头像管理模块**：头像上传、获取、自动清理功能
