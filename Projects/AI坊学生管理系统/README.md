# AI坊学生管理系统

## 项目概述
AI坊学生管理系统是厦门工学院人工智能创作坊的综合管理解决方案，提供学生信息管理、分时段签到、数据可视化等功能。

## 项目结构

### 前端
- **技术栈**：Vue3 + Element Plus + ECharts
- **核心功能**：用户认证、分时段签到、数据可视化、响应式设计
- **详细信息**：查看 [前端README](前端/README.md)

### 后端
- **技术栈**：Spring Boot + MySQL + Redis
- **核心功能**：学生信息管理、签到系统、安全认证、数据统计
- **详细信息**：查看 [后端README](后端/README.md)

## 核心功能
- 学生信息管理
- 分时段签到系统
- 数据可视化看板
- 安全认证与权限管理
- 积分与头像管理
